# V-CRAFT | Digital Portfolio

A sleek, modern, and high-performance digital portfolio showcasing the projects and skills of a B.Tech student at Amrita Vishwa Vidyapeetham. This project focuses on visual excellence and technical efficiency through a "zero-script" architecture, utilizing modern CSS for all interactions and animations.

## Project Overview

V-CRAFT is a personal portfolio website designed to be both a digital resume and a demonstration of modern web design capabilities. The site features a premium aesthetic with a "Dark/Crimson" theme and smooth transitions.

### Key Features
- **Zero-JavaScript Architecture:** All animations, including the typing text effects and scroll-linked corner accents, are handled purely through modern CSS (animation-timeline, :has(), etc.).
- **Dynamic Splash Screen:** A custom entry animation that introduces the brand before revealing the content.
- **Project Showcase:** Detailed project pages for Attendance Management, IoT Thermometers, Thermal Analysis, and DC Protection.
- **Modern Typography:** Uses Google Fonts (Outfit) for a premium feel.
- **Responsive Design:** Optimized for various screen sizes, ensuring accessibility across devices.

## Technical Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3 (Modern/Advanced):** 
  - Glassmorphism effects (backdrop-filter)
  - Scroll-Linked Animations (animation-timeline)
  - CSS Selection state triggers (:has)
  - Keyframe animations for "typing" effects
- **Icons:** Phosphor Icons library.

## Setup Instructions

This project is built using standard web technologies and requires no complex build tools or dependencies.

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/MsximusWrathX/project.git
   ```
2. **Navigate to the Project Directory:**
   ```bash
   cd v-craft-portfolio
   ```
3. **Download Assets:**
   Ensure all image assets (DP.jpeg, bg.jpeg, attendance.jpg, etc.) are present in the root directory.

## How to Run the Project

Since this is a static website, you do not need a server to view it, although using a local server is recommended for better performance and to avoid CORS issues with certain assets.

### Option 1: Direct Execution
Simply open index.html in any modern web browser (Chrome, Edge, or Safari 17+ recommended for full animation support).

### Option 2: Using VS Code (Recommended)
1. Open the project folder in **VS Code**.
2. If you have the **"Live Server"** extension installed, click **"Go Live"** in the bottom-right corner.
3. The site will automatically open in your default browser at http://127.0.0.1:5500/.
